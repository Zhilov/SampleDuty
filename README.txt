* Зачем?
 Приложение создано для быстрого взаимодействия с Jira. При запуске позволяет выбрать шаблон из списка. По нажатию на кнопку открывается Jira с уже вбитыми данными.

* Как установить?
 В папке SampleDuty скриптом есть word-инструкция по установке скрипта в 3cx.

* Как это работает?
 В папке SampleDuty/res есть три файла с расширением txt. Это: data.txt, name.txt, phone.txt. 
	data.txt содержит в себе информацию о каждом шаблоне. В каждой строке располагается текст одного шаблона.
	name.txt содержит в себе лишь одну строку - ФИО оператора. Введите ваши ФИО в первую строку.
	phone.txt содержит в себе входящий номер телефона, в изменении не нуждается, batch скрипт самостоятельно вбивает номер.
	
* Как добавлять новые шаблоны?
 Файл data.txt - база данных шаблонов. Каждая строчка - один шаблон. Сегрегация данных происходит путём добавления точки с запятой (;)в конце каждой фразы или параметра.
 Один шаблон содержит в себе пять полей (соответственно, каждая строка должна содержать пять точек с запятой).Тип шаблона (1 - консультация, 2 - инцидент), название шаблона (будет отображаться на кнопке), Тема, Описание,
 Решение для пользователя (только для консультаций, в инцидентах не нужно добавлять, сразу добавив вторую, закрывающую точку с запятой),
 параметр обращения (если это консультация, указываете true, если инцидент - false).
 Примеры шаблонов:
	Консультация: 
		1;Запись к врачу;Пользователь хочет записаться к врачу;Пользователь хочет записаться к врачу;Рекомендовано обратиться в регистратуру;true;

	Инцидент:
		2;Не поступает контрольный звонок;Не поступает контрольный звонок пользователю;Пользователю при восстановлении доступа не поступает контрольный звонок. Номер активный. Логин от ЛК: (номер телефона). Оператор: (МТС) Время запроса контрольной строки: (время).;
